Practice Problems

1. Consider the following statement:

q ∧ r ∧ p'

What would the truth-table for this statement be?


2. Consider the following statement:

p ∨ q' ∨ r'

What would the truth-table for this statement be?


3. Consider the following statement:

r' ⇒ (q ∧ p')

What would the truth-table for this statement be?


4. Consider the following statement:

(q ∨ p') ⇒ r'

What would the truth-table for this statement be?
