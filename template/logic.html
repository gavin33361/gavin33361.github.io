What is Propositional Logic?

A proposition is a statement that is either true or false, but not both simultaneously.
Examples of propositional statements:

"Molly has posted her holiday photos on flickr." - This will be either be true or false depending on what Molly did.

"Fred will always accept a Facebook friend request from a friend of a friend." - This will be either true or false, depending on Fred's policy in this matter.

"All even numbers are integers." - This is true.

"All integers are even numbers." - This is false.

Is this a proposition?

"This statement is false." - If this statement is true, it would be a contradiction. If this statement is false, it would also be a contradiction. Since this can be neither true nor false, it is not a proposition. This is known as the liar paradox.

We can represent propositions by using logical variables.
For example:

p = "A New York City subway fare is $2.25."
q = "It will rain by the end of the day."
r = "All even numbers are integers."
