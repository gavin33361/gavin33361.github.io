Truth Tables

A logical statement having n component statements will have 2n rows in its truth table. For example, the expression p ∧ q' will have 4 rows.

pqq'p ∧ q'
TTFF
TFTT
FTFF
FFFF

A Boolean expression that is always true is said to be a tautology, whereas a Boolean expression that is always false is said to be a contradiction. Examples:

This is a tautology.

pp'p ∨ p'
TFT
FTT

This is a contradiction.

pp'p ∧ p'
TFF
FTF
