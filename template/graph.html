Logical Graphs

A logical graph is a type of diagram that displays relationships between or among variables through graphical syntax.

Some situations that can be visualized with logical graphs include:


- Communications on Social Networks, such as Facebook

- Connected Cities in a Nation

- Tasks on a "To-Do" List

- Electrical Connections and Wirings

- The Internet



The set "V" represents the set of vertices, while "E" represents the set of edges.

{v, w} ∈ E means that v, w and the set of vertices are connected.


A logical graph can be considered directed, or a digraph, if the edge from one vertex to another can ONLY be traversed in that direction. Otherwise, these graphs are undirected.

For directed graphs, the edge connecting v, w ∈ V is (v,w).

For undirected graphs, the edge connecting the distinct and specific v, w ∈ V is {v,w}.
