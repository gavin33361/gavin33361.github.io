Propositional Forms

Through Boolean variables, we can create Boolean expressions (sometimes called propositional forms). For example starting with Boolean variables p and q, we can get Boolean expressions such as p', q', p ∧ q, p ∨ q, p ∧ q', and so forth.

To define Boolean expression:

Any Boolean variable is a Boolean expression.

If P is a Boolean expression, then P' is also a Boolean expression.

If P and Q are Boolean expressions, then so are P ∧ Q, P ∨ Q, P ⊕ Q, P ⇒ Q, and P ⇔ Q.

Nothing else is a Boolean expression.

Simplified precedence rules for Boolean expressions:

Parenthesized subexpressions come first.

Negation(') comes next.

Multiplicative operation(∧) is donebefore additive operations(∨,⊕).

Use parentheses if you have any doubt. Always use parentheses if you have multiple conditionals.

Evaluate ties left-to-right.

Examples:

(p ∧ q) ∨ r can be simplified to p ∧ q ∨ r.

((p ∧ q) ∨ (p ∧ (r'))) ∧ (p ∨ r) can be simplified to (p ∧ q ∨ p ∧ r') ∧ (p ∨ r).
